SECTION 2 — AI CHAT CREDIT SYSTEM
════════════════════════════════════════════════════════

ZenAvatarChat.tsx is your most expensive feature (API costs).
A credit system controls spend and creates a natural upsell path.

Credit Model:
  • 1 Credit = 1 AI message exchange (user message + AI reply)
  • Credits reset on billing date (not calendar month)
  • Unused credits do NOT roll over (creates urgency)

Credit Allocation per Tier:
  ┌──────────────┬──────────────┐
  │  Plan        │  Credits/mo  │
  ├──────────────┼──────────────┤
  │  Free        │  10          │
  │  Silver      │  100         │
  │  Gold        │  500         │
  │  Platinum    │  Unlimited   │
  └──────────────┴──────────────┘

Top-Up Packs (à la carte, one-time purchase):
  • Starter Pack  — 50 credits for ₹49
  • Booster Pack  — 200 credits for ₹149
  • Power Pack    — 500 credits for ₹299
  • Mega Pack     — 2000 credits for ₹999

Implementation Note:
  • Add a `aiCreditsRemaining` field to the User model
  • Deduct 1 credit per message sent via ZenAvatarChat
  • Show a real-time "X credits remaining" badge in the chat UI
  • When credits hit 0, show an upgrade/top-up modal instead of the input field
  • Track credit purchases in a `CreditTransaction` collection

SECTION 3 — THERAPY SESSION MONETIZATION
════════════════════════════════════════════════════════

Therapy is booked via TherapyHub.tsx / TherapistDashboard.tsx.
This is likely your highest-value revenue stream.

Models:
  1. Pay-Per-Session (default, no subscription required)
     • 30-min session — ₹500–₹800 (therapist sets price)
     • 60-min session — ₹900–₹1,500
     • Platform takes a 15–20% cut per session (configurable in Admin)

  2. Session Bundle Packs (pre-purchased, discounted)
     • 4-session pack  — ₹2,500 (save ₹200–₹500 vs pay-per-session)
     • 8-session pack  — ₹4,500
     • 12-session pack — ₹6,000

  3. Subscription-Included Sessions (see Section 1)
     • Gold: 1 free session/month included
     • Platinum: 2 free sessions/month included

  4. Therapist Revenue Share Model
     • Therapist sets their own base price
     • ZenMind charges 15% platform fee on every session
     • Therapist receives 85% of the session fee, paid out weekly/monthly
     • Admin dashboard already has TherapistsManagement.tsx to manage this

Implementation Note:
  • Add `platformFeePercent` and `therapistRate` fields to the Session model
  • Create a `Payout` model to track therapist payouts
  • Admin dashboard shows revenue breakdown (already has AdminAnalytics.tsx)

SECTION 4 — WELLNESS STORE MONETIZATION
════════════════════════════════════════════════════════

WellnessStore.tsx + WellnessStoreAdmin.tsx are already built.
This is the most "ready to launch" monetization feature.

Revenue Streams:
  1. Paid Asset Downloads (already stubbed, just needs real payment)
     • Guided meditations, worksheets, CBT exercises, journals — ₹49–₹499
     • One-time purchase, permanently accessible after payment
     • Platform keeps 100% (admin-uploaded assets) or revenue-shares with
       therapist-contributed content (70/30 split)

  2. Bundle Deals
     • "Stress Relief Pack" — 5 assets bundled at 30% off
     • "Sleep Hygiene Kit" — 3 assets bundled
     • Admin can create bundles in WellnessStoreAdmin.tsx

  3. Subscription Discounts (already planned in Section 1)
     • Silver: 10% off all paid assets
     • Gold: 20% off
     • Platinum: 30% off (lifetime discount badge)

  4. Therapist-Uploaded Premium Content
     • Therapists can upload their own resources via TherapistDashboard.tsx
     • Admin approves, content goes live in the store
     • Therapist earns 70% of each sale; ZenMind keeps 30%

SECTION 5 — WELLNESS PROGRAMS MONETIZATION
════════════════════════════════════════════════════════

WellnessProgramsUser.tsx + WellnessProgramsAdmin.tsx are built.
Programs are structured, multi-step guided journeys.

Revenue Streams:
  1. Freemium Gate
     • Free tier: access 1 program (admin-designated "free" program)
     • Silver: 2 programs
     • Gold/Platinum: all programs

  2. Standalone Program Purchase (one-time)
     • Users can buy a specific program without subscribing
     • e.g., "14-Day Anxiety Relief Program" — ₹799 one-time
     • Permanently unlocked after purchase
     • Admin marks programs as "free", "subscription", or "purchasable"

  3. Program Completion Certificates
     • On completing a program, user gets a downloadable certificate
     • Could be gated behind Gold/Platinum or sold separately (₹49)

PeerCircles.tsx is already built with community groups.

Revenue Streams:
  1. Circle Creation (Free tier can only join, not create)

     • Mental health NGOs, pharma companies, or wellness brands can
       sponsor a public circle (admin-approved)
     • Flat sponsorship fee ₹5,000–₹20,000/month
     • Sponsor gets a "Supported by" badge in the circle

SECTION 11 — WHAT STAYS FREE ALWAYS (Trust Builders)
════════════════════════════════════════════════════════

Mental health platforms must build trust before monetizing.
These should always remain free to avoid backlash:

  ✓ Crisis helpline info / emergency resources (ResourceHub.tsx)
  ✓ Basic mood check-in (at least 5/month)
  ✓ Onboarding flow (OnboardingFlow.tsx)
  ✓ Therapist match quiz (once)
  ✓ Basic community wall viewing
  ✓ At least 1 free wellness program
  ✓ 10 AI chat messages/month (lets new users experience the feature)
